# Validation Commands

## Development & Build
- `npm run dev`: Starts the Next.js development server.
- `npm run build`: Executes the production build pipeline.

## Quality & Linting
- `npm run lint`: Runs ESLint across the codebase.
- `tsc`: Runs the TypeScript compiler to ensure type safety.

## Testing
*Note: Ensure test suites are added as new features are implemented. Run these suites via `npm test` or the appropriate test runner discovered in `package.json`.*
